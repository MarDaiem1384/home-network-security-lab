\# Home Network Security Lab



\## Overview

This project demonstrates practical cyber security skills through the analysis and documentation of a home network environment.



The aim is to simulate entry-level Security Operations Centre (SOC) tasks, including traffic analysis, basic investigation, and security hardening.



\## Objectives

\- Analyse normal network traffic behaviour

\- Identify potential risks or misconfigurations

\- Apply basic network security improvements

\- Document findings in a structured format



\## Tools Used

\- Wireshark

\- PowerShell

\- Windows Networking Tools

\- Git \& GitHub



\## Activities

\- Created a structured lab environment

\- Performed initial network observation

\- Began documenting network behaviour

\- Set up version-controlled investigation workflow



\## Skills Demonstrated

\- Network analysis

\- Security awareness

\- Technical documentation

\- Command-line usage (PowerShell)

\- Version control using Git



\## Planned Improvements

\- Capture and analyse real network traffic using Wireshark

\- Perform basic network scans

\- Develop PowerShell scripts for analysis

\- Create detailed investigation reports



\## Disclaimer

All work is conducted within a personal lab environment on authorised devices only.
